Desarrollo Web en Entorno Cliente (DWEC)

Ciclo formativo: CFGS Desarrollo de Aplicaciones Web (DAW) Unidad Didáctica 2 · Proyecto ABP Herramientas: JS (navegador), prompt, alert, consola

Enunciado (resumen)

Calcula el presupuesto de un encargo según: páginas B/N y Color, tamaño (A4/A3), doble cara, encuadernaciones y urgencia. Aplica el orden Descuento → Urgencia → IVA. Opera internamente en céntimos y muestra a 2 decimales.

  • IVA 21%; A4: B/N 0,05 €·pág, Color 0,20 €·pág; A3 = ×1,5; Doble cara = ×0,9.
  • Encuadernación 3,00 €·ud; Urgencia +15%; Descuento por volumen: ≥30 € → 5%, ≥60 € → 10%.

Consejo: valida entradas (negativos/NaN → 0) y trabaja siempre en céntimos para evitar errores de precisión.

Ejemplo de entrada (vía prompt)

Páginas B/N: 30
Páginas Color: 0
Tamaño: A4
Doble cara: no
Nº de encuadernaciones: 1
Urgente: no

Salida esperada (resumen)

TOTAL: 5.45 €  (Detalle completo en consola)

Abre la Consola del navegador (F12). Este HTML carga el JS externo y te pedirá los datos.
